# Eval Package

A Python package for evaluating model performance and generating critical difference diagrams.

## Features

- Reads operational parameters from JSON configuration
- Processes CSV data files with configurable column mapping
- Calculates performance metrics using the tools package
- Generates ranked performance tables
- Performs Friedman statistical tests
- Creates critical difference diagrams

## Usage

```bash
eval-run --config config.json
```

## Configuration

See `config.json` for an example configuration file structure.
